# Ucitelsky-rozvrhar
Toto je ročníkový projekt, ktorého cieľom je pomôcť pri výrobe rozvrhov tým,
že namiesto hladania požiadaviek vyučujúcich v emailoch,
bude všetko na jednom mieste v prehľadnom formáte.
